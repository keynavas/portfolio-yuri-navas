# 🧱 Modularização: A Arte do "Dividir para Conquistar"

## 📝 Descrição do Projeto
Este projeto explora a **Modularização** como o pilar central da Engenharia de Software moderna. A proposta é demonstrar como transformar códigos monolíticos ("espaguete") em sistemas inteligentes, compostos por unidades independentes de alta performance.

O foco aqui é a **Engenharia de Soluções Lógicas**: entender que um problema complexo não deve ser resolvido de uma vez, mas sim decomposto em partes menores que respeitam a **Lógica Física** do problema e o seu **Contexto** operacional.



## 🚀 Pilares do Pensamento Computacional Aplicados
* **Decomposição:** Fragmentação do sistema em módulos funcionais.
* **Abstração:** Criação de interfaces que escondem a complexidade interna (conceito de Caixa Preta).
* **Engenharia de Contexto:** Funções que se comunicam de forma segura, garantindo que o dado correto chegue ao destino certo.
* **Algoritmos Eficientes:** Passos ordenados dentro de cada módulo para garantir previsibilidade e baixo consumo de recursos.

## 🏗️ Princípios de Design (Checklist do Código Limpo)
Para garantir uma solução lógica robusta, este repositório segue rigorosamente:

* **Princípio DRY (Don't Repeat Yourself):** A lógica nunca é duplicada. Se um cálculo é usado duas vezes, ele se torna um módulo reutilizável.
* **Alta Coesão:** Cada função tem uma "personalidade única" e executa estritamente uma tarefa.
* **Baixo Acoplamento:** Módulos independentes com poucas dependências externas, facilitando a troca ou atualização de partes do sistema sem colapsar o todo.
* **Isolamento de Escopo:** Uso de parâmetros e retornos para evitar a poluição do sistema com variáveis globais desnecessárias.

## 📊 Resultados e Aprendizados
A adoção da modularização permite:
* **Manutenibilidade:** Encontrar e corrigir bugs em minutos, não em horas.
* **Testabilidade:** Cada módulo pode ser testado isoladamente (Teste de Unidade).
* **Clareza Top-Down:** O programa principal conta a "história" do software, enquanto as funções cuidam dos detalhes matemáticos e técnicos.



## 🔧 Como Executar
1. Clone o repositório.
2. Navegue pelos módulos na pasta `/src`.
3. Execute o orquestrador principal: `python main.py`.
